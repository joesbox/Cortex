
# Cortex

Cortex is the configuration software for SynapsePDM, an open source automotive Power Distribution Module, written in Avalonia C#.NET.

It's currently in Alpha stage with limited and/or incomplete features. Beware of messy, sub-optimal code herein.

Further details can be found at https://wiki.joeblogs.uk.

## What works
- Serial communication protocol
- Live data stream from the controller
- Channel, input and system parameter configuration updates to the controller 
- Event log
- Live charting and data visualisation

## What's next
- Configuration save/load
- Telemetry configuration
- IMU/GPS/Tracking configuration
- General code review and clean up
